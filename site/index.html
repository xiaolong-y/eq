eq

A terminal-based task manager for high-performance prioritization.

Get Started GitHub

EQ (Eisenhower Quadrants) is a CLI tool built in Rust that helps you prioritize tasks using the Eisenhower Matrix. It is designed for speed, keyboard-centric navigation, and focus.

Installation

Install directly from source using Cargo:

$ cargo install --path .

Or add the binary to your path manually.

User Guide

1. Adding Tasks

You can add tasks via the CLI or inside the TUI. The parser understands natural language for priority.

# High Urgency (!!!) and High Importance ($$$) -> Q1
$ eq add "Fix production bug !!!$$$"

# Explicit numerical priority (u=urgency, i=importance)
$ eq add "Read documentation u1i3"

2. The TUI Interface

Launch the interactive mode with eq tui. The interface is divided into 4 quadrants.

3. Zen Mode

Press z to enter Zen Mode. This isolates the current task and starts a Pomodoro timer with floating particles for focus.

4. AI Chat

Press c to open the AI chat. You can ask the AI to help you prioritize or break down tasks.

Note: Requires OPENAI_API_KEY in your .env file.
